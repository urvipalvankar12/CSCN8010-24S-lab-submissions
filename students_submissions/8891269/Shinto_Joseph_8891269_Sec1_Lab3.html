Lab3 - Univariate Linear Regression on the Scikit-Learn Diabetes Dataset

Step 1: Frame the Problem and Look at the Big Picture

Objective: Predict disease progression (a quantitative measure one year after baseline) using Body Mass Index (BMI) as the independent variable.

Problem Type: Supervised learning, regression task.

Performance Metric: Mean Squared Error (MSE) will be used to evaluate the model performance.

Step 2: Obtain the Data

We'll utilize Scikit-Learn's datasets module to load the necessary data.

In[19]:
from sklearn import datasets
import numpy as np

# Load the dataset
data_x, data_y = datasets.load_diabetes(return_X_y=True)

# use the Body mass index
index_col = 2
data_x_bmi = data_x[:, np.newaxis, index_col]

Step 6: Optimize the Model

Divide the data into training and testing sets, train the model, and assess its performance.

In[21]:
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

# spliting data into test and train data
data_x_train, data_x_test, data_y_train, data_y_test = train_test_split(data_x_bmi, data_y, test_size=0.2, random_state=42)

# creating linear regression model
model = LinearRegression()

# train the models
model.fit(data_x_train, data_y_train)

# predict
data_y_pred = model.predict(data_x_test)

# mean squared error
mean_sq_err = mean_squared_error(data_y_test, data_y_pred)

print(f'Model slope (coefficient): {model.coef_[0]}')
print(f'Model intercept: {model.intercept_}')
print(f'Mean Squared Error: {mean_sq_err}')
